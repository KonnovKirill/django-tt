Это Django-приложение для управления финансовыми транзакциями с поддержкой категорий, подкатегорий, типов и статусов.

## Установка зависимостей

Проект использует Django и другие Python-пакеты. Убедитесь, что у вас установлен Python 3.8+.

1. Создайте виртуальное окружение (рекомендуется):
   ```
   python -m venv venv
   venv\Scripts\activate  # На Windows
   # source venv/bin/activate  # На Linux/Mac
   ```

2. Установите Django и необходимые пакеты:
   ```
   pip install django
   ```
   pip install -r requirements.txt
   ```

## Настройка базы данных

Проект использует SQLite по умолчанию (файл `db.sqlite3`).

1. Перейдите в директорию проекта (где находится `manage.py`):
   ```
   cd finance
   ```

2. Примените миграции для создания таблиц в базе данных:
   ```
   python manage.py migrate
   ```

3. (Опционально) Создайте суперпользователя для доступа к админке:
   ```
   python manage.py createsuperuser
   ```
   Следуйте инструкциям для ввода имени пользователя, email и пароля.

## Запуск веб-сервиса

1. Запустите сервер разработки Django:
   ```
   python manage.py runserver
   ```

2. Откройте браузер и перейдите по адресу: `http://127.0.0.1:8000/`

3. Для доступа к админке: `http://127.0.0.1:8000/admin/` (используйте учетные данные суперпользователя).

## Дополнительные замечания

- Если вы изменяли модели, не забудьте создать и применить миграции:
  ```
  python manage.py makemigrations
  python manage.py migrate
  ```
- Для остановки сервера нажмите `Ctrl+C` в терминале.
- Если возникают ошибки, проверьте версию Python и установленные пакеты.

## Структура проекта

- `finance/`: Основной каталог Django-проекта
- `transactions/`: Приложение для управления транзакциями
- `templates/`: HTML-шаблоны
- `static/`: Статические файлы (CSS, JS)
